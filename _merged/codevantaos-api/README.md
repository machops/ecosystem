# AutoEcoOps API

本專案展示了 GitHub 的最佳實踐，包含自動化工作流程和智能任務分配系統。

This repository demonstrates GitHub best practices, including automated workflows and intelligent task assignment.

## 功能特色 (Features)

- 🤖 **自動分配任務 (Automatic Task Assignment)**: 當新的 Issue 或 Pull Request 被建立時，系統會根據標籤、檔案變更和關鍵字自動指派給最合適的維護者。
- 📄 **HTML 驗證 (HTML Validation)**: 自動驗證 HTML 檔案的正確性。
- 🎨 **Primer CSS**: 使用 GitHub 的 Primer CSS 框架。

## 文件 (Documentation)

- [自動分配任務說明 (Auto Assignment Documentation)](docs/AUTO_ASSIGN.md) - 了解自動指派的工作原理和自訂規則

## 使用方式 (Usage)

建立新的 Issue 或 Pull Request 時，系統會自動根據以下規則進行指派：
- 標籤規則 (Label-based rules)
- 檔案變更規則 (File-based rules for PRs)
- 關鍵字規則 (Keyword-based rules)
- 預設規則 (Default fallback)

詳細資訊請參閱 [自動分配任務文件](docs/AUTO_ASSIGN.md)。
